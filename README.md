SkidScan- Dark web Monitor & Web Forensics
Dark Web Monitoring & Web Forensics Tool

A Windows-based forensic tool to monitor dark web activity and analyze suspicious web content for cybercrime investigation.


🔍 What is SkidScan?
SkidScan is a group cybersecurity project focused on dark web crawling and web forensics. It assists investigators in monitoring dark web sources, collecting digital evidence, and supporting cybercrime investigation workflows.

⚙️ What It Does

Dark Web Monitoring — Tracks and flags suspicious dark web activity
Web Forensics — Collects and analyzes web-based evidence
Data Collection — Gathers threat indicators for investigation
Investigation Support — Assists in documenting cybercrime cases
